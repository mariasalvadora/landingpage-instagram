## Como baixar

``` bash
# Clonar o repositorio
 $ git clone https://github.com/mariasalvadora/landingpage-instagram.git

 # Entrar no diretório
 $ cd landingpage-instagram

 # Instalar
 $ liveserver 

 # Abrir no browser
 $ botão direito(index.html) - Open with live server
```
